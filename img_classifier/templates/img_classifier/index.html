Image Classifier

This Computer Vision projects uses pre-trained model VGG16 to classify user uploaded images between a total of 1,000 clases. VGG model uses Convolutional Neural Networks (CNN) to extract features from an approximate of 1.2 million training images. Along with the prediction label, you can see the probability of the image corresponding to the displayed label.

Implemented through Keras framework, images hosted in AWS S3 bucket.
